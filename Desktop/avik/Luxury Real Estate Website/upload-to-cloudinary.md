# Upload Video to Cloudinary

1. Go to https://cloudinary.com (Free account)
2. Upload your bg_easymakan.mp4 file
3. Copy the video URL (will look like: https://res.cloudinary.com/your-cloud/video/upload/v123456/bg_easymakan.mp4)
4. Replace the video src in Hero.tsx with this URL

## Steps:
1. Sign up at cloudinary.com
2. Go to Media Library
3. Click "Upload" 
4. Select bg_easymakan.mp4
5. Copy the generated URL
6. Update Hero.tsx with the new URL
